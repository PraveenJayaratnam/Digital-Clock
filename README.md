# Digital-Clock
Simple Digital Clock with JavaScript.
used HTML5,CSS3,JavaScript and Bootstrap5.
Has an abilty to update 2 times within a second.
It wrap single digits(0-9) with another zero to make a double digit number.
12hr format clock.
